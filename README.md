# Autonomous Pan-Tilt Tracking Camera

## Overview
The AI Pan-Tilt Tracking Camera is an embedded vision system that automatically detects and tracks moving objects using a two-axis pan-tilt mechanism. An Arduino controls the servo motors while the camera system provides visual feedback for autonomous tracking.

## Features
- Autonomous object tracking
- Pan and tilt servo control
- Arduino-based control system
- Camera integration
- Expandable to mobile robotics

## Hardware
- Arduino UNO
- ESP32-CAM or XIAO ESP32S3 Sense
- 2 Servo Motors
- Pan-Tilt Bracket
- External 5V Power Supply

## Software
- Arduino IDE
- C++
- Serial Communication
